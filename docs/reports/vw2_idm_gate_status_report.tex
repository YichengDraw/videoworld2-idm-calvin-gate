\section*{Verdict}

The current repository supports an offline CALVIN gate and a mock-only closed-loop smoke path. It still does not contain a valid real CALVIN closed-loop adjudication. The real closed-loop path is blocked by two concrete facts: the repository evaluator rejects non-mock datasets, and the local rescue bundle does not contain raw CALVIN frame/action files needed by the dataset loader.

\section*{Reliability Changes}

\begin{itemize}
  \item The latent planner now uses a causal decoder mask, preventing teacher-forced target tokens from leaking future labels into earlier predictions.
  \item Latent caches and window indexes now carry validation metadata and content digests; saved window payloads are re-derived before reuse, and stale specs, stale adapter checkpoint contents, stale split identity, stale full-span CALVIN frame provenance, duplicate cache keys, or key-set mismatch now fail closed.
  \item Official dLDM tokenizer checkpoints are rejected unless they cover the encode path used for cache extraction and predicted-code embeddings.
  \item Train and validation manifests are checked for shared episode IDs, shared file-backed episode sources, byte-level and semantic copied file contents, copied or partially overlapping CALVIN frame contents across or within splits, and overlapping CALVIN frame spans before dataset construction.
  \item DataLoader workers and generators are seeded from the experiment seed, and evaluation entrypoints request deterministic CUDA behavior by default.
  \item Direct-policy, MLP, planner, verifier, teacher, resume, and CLI checkpoints are validated against explicit architecture, IDM conditioning metadata, planner/teacher checkpoint identity, and path guards before loading; distillation now preserves predicted-code teacher conditioning.
  \item Verifier-assisted evaluation reloads the verifier's paired state encoder instead of feeding a verifier trained on one encoder with tokens from another encoder.
  \item Cross-directory config inheritance preserves the directory that defined each relative path, and repeated relative path strings are resolved by config key path to avoid source-directory collisions.
\end{itemize}

\section*{Artifact Boundary}

The non-versioned rescue bundle contains manifests, window indexes, latent caches, checkpoint files, resolved configs, and offline-evaluation outputs. It does not contain raw CALVIN RGB/proprio/action frame files. The rescued manifests still reference the original remote Linux roots, so local replay requires remapping or restoring the raw dataset. The conditional real closed-loop goal remains open until the real CALVIN environment and raw data are available within the run budget policy.

The locally observed \texttt{ldm\_tokenizer\_training\_init\_weights.pt} file is not accepted as a verified official tokenizer for fresh cache extraction because it does not cover all encode-path parameters. Fresh official-cache extraction requires a complete tokenizer checkpoint, or an explicitly diagnostic partial-checkpoint override.
