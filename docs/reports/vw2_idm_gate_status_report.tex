\section*{Reliability Work}

\begin{itemize}
  \item Planner masking prevents teacher-forced future-token leakage.
  \item Data and cache guards validate split identity, window payloads, latent-cache keys, tokenizer coverage, and CALVIN frame-span overlap.
  \item Checkpoint guards validate controller architecture, conditioning mode, planner/teacher identity, and path resolution before load.
  \item Result packaging records provenance, privilege/deployability flags, metric origin, and non-finite metric checks.
\end{itemize}

\section*{Boundary}

\begin{itemize}
  \item Raw CALVIN RGB/proprio/action files are absent from the public repository.
  \item Rescued manifests point to original remote Linux paths; replay needs the restored dataset or explicit path remapping.
  \item Fresh cache extraction needs a complete official dLDM tokenizer checkpoint.
  \item A real verdict needs one missing experiment: fixed-episode CALVIN closed-loop evaluation with identical starts, instructions, camera pipeline, action normalization, clipping, and timeouts across controllers.
\end{itemize}
