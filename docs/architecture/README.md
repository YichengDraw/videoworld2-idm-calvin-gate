# Architecture Diagrams

This folder keeps source diagrams for the public README visuals.

- `model_pipeline.mmd` describes the implemented `videoworld2/robot_idm` model and evaluation data flow.
- `experiment_flow.mmd` describes the reliability cleanup and result-publication flow.

Rendered SVGs are committed under `assets/readme_figs/` so GitHub can display them without a Mermaid build step.
